# CUBICENGINE Studio

マインクラフトのアドオン（統合版）やMOD（Java版）を直感的に作れる、コーディング不要のビジュアル開発環境です。

## 特徴
- **完全ローカル・オフライン動作**: アカウント登録不要で、あなたのPC上で安全に動作します。
- **SPROUT (Bedrock版)**: 統合版のアドオン開発をサポート。
- **GROVE (Java版)**: Java版のMOD開発およびGradle連携ビルドをサポート。
