# つなぎ設計メモ（Phase 2 叩き台）— カードを「場に出す」

> シオン(設計) 2026-06-14。**LogicPanel本体は触らずドキュメントのみ**。
> 前提：世界観=ポケカ寄り(decision_plan_a_livestage)、色=ヒマワリ/中身=シオン(feedback_color_to_himawari)。
> ステータス：**叩き台**。伊波さんの判断が要る所を 🚩 で明示。

## 0. 何を解決するか
今の繋ぎ＝「タップでスロット起動→光った相手をタップ→ケーブル」＝**古いノードグラフ配線**。面倒＆ブロックエディタの名残で、"新規性>慣れ"に反する。
→ **カードゲーム化**：カードは「繋ぐ」んじゃなく「**場に出す**」。並べた順＝実行順。配線ゼロ。

## 1. 中核アイデア（いちばん大事）
**データ構造は今のまま（CBlockグラフ：`nextId`/`thenId`/`elseId`/`innerId`）を再利用する。** 変えるのは"繋ぎ方"のUIだけ。
- 「場に出す」＝末尾カードの `nextId` に新カードを繋ぐ（自動連結）。
- これにより **codegen も LiveStage も無改修**（同じグラフを読むから）。"出した手が即・世界で動く"がタダで付く。
- ＝ Phase 2 は「配置/操作UIの差し替え」で済み、**ロジックの土台は壊さない**（リスク最小）。

## 2. 盤面（プレイマット）
- 中央＝**横一列のレーン**（場）。左＝先頭、右へ伸びる。
- **リードカード（トリガー）**＝レーン先頭の専用枠。1枚。
- 以降、アクションカードが右へ数珠つなぎ。
- 出すたび **「2 CHAIN！ 3 CHAIN！」** のコンボ表示（カードゲームの連鎖快感）。

## 3. 操作
- **出す**：STEP1/2/3で選んで設定 → SPAWN＝カードがレーン末尾にカチッと着地（＝`lastCard.nextId = new.id`）。
- **並べ替え**：レーン上でドラッグ → 順番入れ替え（`nextId`鎖を再計算）。
- **抜く**：場から下に弾く／✕ → 前後を再連結。
- **値**：STEP3スロットで設定済み。カード上では編集しない（既決）。

## 4. 枝分かれ（条件・繰り返し）🚩 ここが一番むずい
`co_if` / `ct_rep` は「中に他のカードを抱える」**器カード**。
- 出すと **「そうなら」ポケット**が開く（カードがカードを持つ＝ポケカの進化/付与的）。
- 以降の"出す"は **アクティブなポケットの中へ** 入る → `thenId`鎖。
- 🚩 **未決**：ポケットから抜けて本線へ戻る操作（"枝を閉じる"ボタン／挿入点カーソルをタップ移動／インデント表示）。＝Cブロック問題。
  - 叩き台の仮案：**器カードをタップで「中に入れる/出る」を切替＋発光で現在の挿入点を示す**。
- `else`（ちがうなら）：器カードに2つ目のポケット。**まず then だけ対応→else は後追い**でも可。

## 5. 値・計算の "アタッチ"（上級・後回し可）
`co_if` の条件(`innerId`)や計算の被演算子は、ポケカの**エネルギー付与**的に「カードにカードを貼る」。
- ただし STEP3スロットで多くの値が埋まるので、**初期は条件カード＝単体で完結するもの**（HP不足/夜/スニーク/雨…）だけ場に出せれば回る。
- `co_and`/`co_or` や calc の入れ子＝上級者向けとして後段で。

## 6. トリガーの扱い
- リード枠は1枚＝1つの「手」。
- 🚩 複数トリガー（複数の手を並行）：レーンを複数にするか、1手1Modにするか。叩き台では **まず1レーン1トリガー** でシンプルに。
- `ct_int`（インターバル）は"チェーンの一手"でなく **リード（トリガー）側** に置くべき（`runInterval`増殖回避。codegen監査で出た件）→ **ct_int はトリガー分類へ変更を提案**。

## 7. 既存資産との対応表
| 概念 | 今のデータ | 「場に出す」UIでの意味 |
|------|-----------|------------------------|
| 実行順 | `nextId` 鎖 | レーンの 左→右 |
| 枝(そうなら) | `thenId` | 器カードのポケット |
| 枝(ちがう) | `elseId` | 2つ目のポケット |
| 条件/値の差込 | `innerId` | カードへのアタッチ |
| 値 | `fields` | STEP3スロット |

5. else / アタッチ / 複数レーン（上級・後段）。
