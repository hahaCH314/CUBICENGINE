# 📦 CUBICENGINE Studio 3Dモデルエディタ操作マニュアル

この説明書では、アドオンやMODの3Dモデルを直感的に作成するためのエディタ（`ModelPanel`）の使い方を、マイクラ風に分かりやすく解説します！

---

## 🛠 1. 画面の構成

エディタ画面は大きく分けて3つのエリアで構成されています。

1. **中央：3Dビューポート（作業エリア）**
   - 3D空間で立方体（キューブ）を組み立てるメインの場所です。
   - 地面にはグリッドが敷かれており、左下には座標軸を示す **[ X（赤） / Y（緑） / Z（青） ]** のインジケーターがあります。
2. **左側：ツールサイドバー**
   - 立方体の追加、削除、複製、ペイントツールなどの基本アクションを行います。
3. **右側：プロパティパネル**
   - 選択した立方体の名前、座標、大きさ、角度、面ごとの色やテクスチャを細かく数値で調整する場所です。

---

## 🖱 2. 基本的なカメラ操作（視点変更）

3D空間を自由に見回すための操作方法です。

* 🔄 **視点の回転（見回す）**
  - **操作**: 3Dエリアを **左クリックしながらドラッグ**
  - 空間全体をくるくると回転させて、モデルをあらゆる角度からチェックできます。
* ✋ **平行移動（カメラ位置の移動）**
  - **操作**: **右クリックしながらドラッグ**、または **`Ctrl`（または `Cmd`）キーを押しながら左ドラッグ**
  - カメラ自体を上下左右にスライドさせます。モデルの端や裏側を細かく見たいときに便利です。
* 🔍 **ズーム（近づく・遠ざかる）**
  - **操作**: マウスの **ホイールスクロール**
  - モデルに近づいて細部を塗ったり、遠ざかって全体像を確認したりできます。

---

## 🧱 3. モデルの作成と編集

### 1. 立方体（キューブ）の追加・複製・削除
- **追加**: 左のツールサイドバーの **`＋`（追加ボタン）** をクリックすると、原点付近に新しい立方体が出現します。
- **複製**: 立方体を選択した状態で、ツールサイドバーの **`◷`（複製ボタン）** またはキーボードの **`Ctrl + D`** を押すと、隣の位置に全く同じ立方体がコピーされます。
- **削除**: 不要な立方体を選択し、ツールサイドバーの **`🗑`（ゴミ箱ボタン）** または右パネルの **`🗑 このブロックを削除`** ボタンを押すと削除できます。

### 2. 立方体の選択
- **単一選択**: 立方体を直接左クリックします。選択されると外枠がテーマカラー（ポーションピンク）に光ります。
- **複数選択**: キーボードの **`Ctrl`（または `Cmd`）キーを押しながら** 立方体を次々とクリックしていくと、複数の立方体をまとめて選択し、同時に移動やグループ化ができます。

### 3. プロパティの調整（右パネル）
選択した立方体の形状を数値で精密にカスタムします。
* **位置 (X / Y / Z)**: 空間内での配置場所です。
  - 💡 **グリッドスナップ**: 「有効にする」にチェックを入れると、`0.5` や `1.0` 刻みでカチカチと吸着するように移動できるようになり、配置が圧倒的に楽になります！
* **大きさ (X / Y / Z)**: 立方体のサイズです。引き伸ばして「薄い板（`0.1`）」や「長い柱（`5.0`）」を作ることができます。
* **回転 (度)**: `5`度刻みで傾けることができます。キャラクターの手足の角度をつける際などに活躍します。

## 🎨 4. 色付けとテクスチャ（見た目の変更）

### 1. 面ごとの色設定
- 右パネルの **「面の色」** にある6つの四角（`top` / `bottom` / `front` / `back` / `left` / `right`）をクリックすると、カラーピッカーが開いて面ごとに個別の色を設定できます。

### 2. 独自テクスチャ（PNG画像）の貼り付け
- **「テクスチャ（PNG）」** エリアから、面ごとにPC内の画像をアップロード（📤）して貼り付けることができます。
- アップロードされた面には自動的に画像が反映され、ドット絵の質感が綺麗に表現されます（ドットがぼやけない NearestFilter 処理が施されています）。

### 3. テクスチャプリセット
- 「草」「石」「金」「ダイヤ」などのプリセットボタンを押すと、マインクラフトおなじみの配色パターンを一発で適用できます。

### 4. ペイントモード（直感カラーリング）
- ツールサイドバーの **`✎`（ペンボタン）** を押すと、**ペイントモード** に切り替わります。
- このモード中は、右パネルで色やテクスチャを選択した状態で、**3D空間の立方体の面を直接クリックするだけ** で、狙った面を瞬時に塗りつぶすことができます！
- 終わるときは、もう一度 `✎` ボタンを押すか、**`Esc` キー** を押すと通常の選択モードに戻ります。

---

## 💡 プロの裏技とコツ
1. **グループ機能を活用しよう**:
   複数選択した立方体を「新しいグループ作成」でグループにまとめると、パーツ（例：「頭」「右足」など）ごとに管理しやすくなり、作業効率が上がります。
2. **アイテムとブロックの切り替え**:
   画面上の **`⊞ ブロック`** と **`✨ アイテム`** ボタンで、作成ターゲットを切り替えられます。アイテムモードでは最初からコンパクトなサイズ（`0.5`）で追加され、地面にぴったり接地します。
