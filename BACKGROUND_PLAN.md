# ロジック画面 背景リニューアル 計画書（ヒマワリ向け）

作成: シオン / 2026-06-12 / 依頼: 伊波さん
対象: `app/editor/LogicPanel.tsx` の `ThemeBackdrop`（背景レイヤー）まわり **のみ**

---

## 0. このタスクの一言

ロジック画面の背景が今「真っ暗な空間＋裸電球1個」で寂しい。
**背景だけ**を作り込んで、アナログ＝工房／デジタル＝電脳 の世界観を立たせる。
ブロックや配線などの**ロジックには一切触らない**。

---

## 1. 絶対制約（厳守 / 触禁）

1. **背景レイヤー（`ThemeBackdrop` と、その描画に使うCSS/SVG/canvas）だけを変更する。**
   - ブロック・スナップ・配線・armed接続・パン計算・床クリップ・bump/snap アニメ等の**ロジックは一切触らない**。
   - トレイ／ヘッダ／HOW TO PLAY／クレジット等、背景以外のUIも変更しない。
2. **ローカル（オフライン）エディション。** 外部CDN・外部画像URL・Webフォント取得などは**禁止**。
   背景は CSS グラデ / インライン SVG / canvas / コードで生成した図形で**自前描画**する。アセットを足すならリポジトリ内に同梱。
3. **ブロックの可読性が最優先。** 背景は両テーマとも**暗め**に沈め、ブロックの文字・色が常に読めること。背景が主張してブロックを食ったら失敗。
4. **両テーマ必ず zoom / pan に追従させる**（§4 参照）。これは必須要件。
5. ビルド健全性: `npm run build` が TS エラーなしで通ること。`npm install` は **`--legacy-peer-deps` 必須**。

---

## 2. アナログ（工房）の絵

コンセプト: **「作り手の作業部屋」**。マイクラの要素は欲しいが、ベタなテクスチャ流用ではなく**このアプリ独自のオリジナル感**を出す（角を丸める／板目を手描きにする／色味を独自に等）。

- **裸電球は主役で残す**（吊りコード＋暖色グロー＋光の輪）。今のが活きてるので壊さない。
- **奥の木壁**: 横張りの板目。暗く、電球の光が当たる中央だけ明るく浮かぶ。マイクラのオーク板"風"だがドット解像度を上げ過ぎず、手描きの板目で独自感。
- **真鍮の配管**が壁を横切る。電球の光をうっすらハイライト（金属反射）。
- **作業台／棚の縁**を画面下〜端にちらっと（部品箱・瓶などの影）。マイクラの作業台モチーフを抽象化。
- **ホコリ粒子**を光の輪の中だけ、ごく少数ゆっくり漂わせる（シンプルなアニメ。無理なら省略可）。
- 全体に**暖色ビネット**。中央＝電球の光だけ世界がにじむ。

NG: 明る過ぎ／壁テクスチャが主張し過ぎ／彩度の高い色を散らす。あくまで「暗い部屋＋灯り」。

---

## 3. デジタル（電脳）の絵

コンセプト: **「オーガニックグリーンのホログラム ＝ 自然とメタバースの融合」**。電脳世界だが**派手にしない**。1色（緑系）で統一した静かなホログラム投影に、ラメ（きらめき）の質感を散らす。

**色（確定）**: オーガニックグリーン＝葉/苔寄りの落ち着いた緑。**ネオンの原色緑（`#00ff66`）にはしない**。

- ベース単色のアンカー: `#4fd98a` 前後（実機で微調整可。彩度を落とした自然な緑）。
- **ラメ**: 同じ緑系の明るいミント `#cffbe4` の極小スパークルが、ホログラム線上をゆっくり漂う。ごく稀に金の微粒子を一粒だけ混ぜて“ラメ”の質感を出す。**あくまで控えめ**（派手にしない指定を維持）。
- 地色は**ほぼ黒〜濃い地色**。色は上記の緑系で**統一**（虹色・多色・派手なネオン乱舞は禁止）。

絵づくり:

- **ワイヤーフレームのグリッド床** か **簡素なホログラム投影面**を緑の低彩度グローで。
- **立方体ブロックの輪郭（ワイヤーフレーム）**＝メタバース要素が奥にうっすら浮かぶ（マイクラ要素も兼ねる）。
- そこに**蔓・葉の有機的カーブ**をうっすら絡める＝自然要素。立方体（人工）×有機曲線（自然）の融合が今回の肝。
- **走査線（スキャンライン）**をうっすら、ゆっくり流す。
- グローは控えめ。にじみとラメで“生きた電脳”を出すが、まぶしくしない。

NG: 原色ネオン緑／多色／強い発光／パーティクル乱舞／攻殻ガチ盛り。**静かな単色＋ささやかなラメ**が今回の指定。

## 4. zoom / pan 追従（必須・両テーマ共通）

直近コミット `6703a8e` で `ThemeBackdrop` は既に `zoom`/`pan` を受け取っている。これを使って背景を**動かす**。

- 背景は **視差（パララックス）追従**にする＝奥のレイヤーほどゆっくり、手前ほど大きく動く。これで「ズーム対応」と「奥行き」を同時に満たす。
  - 例: 奥の壁/グリッド床 = 係数小（pan*0.2 など）、手前の配管/作業台縁 = 係数大（pan*0.5 など）。数値は実機で調整。
- ズーム時にレイヤー端が見切れないよう、各レイヤーは**画面より大きめに敷く（オーバースキャン）**。
- zoomToFit（⊡）・リセット（⊙）・縦横パン・ズームイン/アウトの**全操作で破綻しない**こと。

> 「画面完全固定」には**しない**。伊波さん指定で必ずズームに連動させる。

## 5. ブロックのアニメ（シオン担当可・テーマ連動）

> 伊波さん指定: **面白さが出るならシオンが直接やってよい。** 背景（ヒマワリ）と並行で進める別ステップ。やり過ぎ厳禁、テーマと連動させて「灯る／流れる」を軸に。

候補（シンプルだが面白い、背景と絡める）:

- **アナログ**: 掴んで離すと作業台に置くように *ことっ* と着地＋影が一瞬伸びる。接続成立で電球がチカッと反応（光の輪が一瞬広がる）＝「回路に灯がともる」。
- **デジタル**: 接続成立で配線を単色ホログラムの光パルスが流れる。ブロック追加時に走査線がそのブロックを一度なめる（マテリアライズ風）。
- **共通**: アイドルでごく僅かな浮遊（上下1〜2px・ゆっくり）。過剰なら不要。

担当分担: 背景＝ヒマワリ。ブロックアニメ＝シオンが入れる場合あり（このセクションはシオンが触る前提なので、ヒマワリはブロックのアニメ実装には着手しなくて良い）。重複実装で上書きしないこと。

## 9. 【追補 2026-06-12】デジタル主役ホログラム ＝ ヒマワリ担当（SVGアートのみ）

> 分担: **器（位置・サイズ・パララックス・重なり）＝シオンが確定済み。ヒマワリは中の `<svg>` アートだけ差し替える。** 器のCSS（位置/transform/zIndex/グロー）は触らないこと。アナログの電球に対応する「電脳の主役＝光源」。

### 編集場所（ここだけ）
`app/editor/LogicPanel.tsx` → `CyberBackdrop` 内、コメント `主役ホログラム：奥中央に浮かぶ…` の div の中の **`<svg viewBox="0 0 120 120" …>` の中身だけ**を描き直す。

### 器の確定スペック（変更禁止・参考用）
- 位置: `left:"50%" top:"40%"`（キャンバス中央やや上＝アナログの電球と同じ吊り位置）。
- パララックス: `translate(calc(-50% + pan.x*0.18), calc(-50% + pan.y*0.15)) scale(1 + (zoom-1)*0.25)`（背景奥行きプレーン。ブロックより控えめに追従）。
- 重なり: `zIndex:3`（ブロックより必ず奥）。背後に緑グロー（`cyber-pulse`）あり。
- アニメ: 器に `holo-float`(ゆるい上下) ＋ svgに `holo-glow`(opacityの明滅)。このアニメは維持でOK。

### 描いてほしい絵（中身）
- モチーフ: **蔓の絡む発光ワイヤーフレーム立方体**（= あなたの作ったムードボード `cyber_hologram_*.png` の中央の立方体＋葉/蔓）。マイクラのブロック × 自然の融合。
- ビューポート: `viewBox="0 0 120 120"`、表示サイズ ~260px 角（器側で指定済み。最大 ~320 まで。中央の作業エリアを潰さない範囲）。
- 色: **モノクロの緑family のみ**。線=`#5fe3a0`、蔓=`#85ebba`、葉の塗り=`rgba(133,235,186,0.15前後)`、奥のエッジ=`#3fae7d`。**他の色は使わない**（金のラメは別レイヤーで既にあるので不要）。
- 線の `opacity` は **0.5以下**（半透明＝ホログラム。ブロックの可読性最優先で背景に沈める）。strokeWidth 0.8〜1.4 程度。

### 厳守
- **ラスター画像（png等）を貼らない。** SVG/CSSで自前描画（ローカル/オフライン制約）。
- 派手にしない（強い発光・多色・うるさいアニメ NG）。あくまで奥で静かに浮かぶ。
- 器のdivの style（位置/transform/zIndex）は触らない。`<svg>` の中身だけ。
- 迷ったらシオン経由で伊波さんに確認。
